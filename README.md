# dotnetcli
Create an ASP.NET Core project using Aurelia and its Toolbelt

Install the template by running:
```shell
  dotnet new --install AureliaToobelt.AspNetCore::*
```

then simply run: 

```shell
  dotnet new aut
```

Bear in mid to restore required packages:

```shell
  dotnet restore  
```
```shell
  yarn install
```

Run the project:

```shell
  dotnet run
```
